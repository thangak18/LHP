# Slide web lộ trình thuật toán Level 0-3

Web tĩnh HTML/CSS/JS thuần, trình bày các module trong roadmap:

- Level 0: nhập xuất, vòng lặp, mảng, xâu, hàm, truy vấn.
- Level 1: vector, đệ quy, backtracking, pair/struct, greedy, map, set, sorting, prefix sum, two pointer, math.
- Level 2: binary search, coordinate compression, stack, queue/deque/priority queue, chia đôi tập, Math_2.
- Level 3: DP, KMP, BFS/DFS, segment tree, inclusion-exclusion, knapsack, LIS, LCS, grid DP, range DP, chia căn, hashing, shortest path.

Mỗi chủ đề có định nghĩa, lý thuyết trọng tâm, ví dụ, mã giả và code C++17 mẫu.

## Chạy local

Mở trực tiếp file:

```text
index.html
```

## Deploy GitHub Pages

1. Push repo này lên GitHub.
2. Vào `Settings` -> `Pages`.
3. Chọn `Deploy from a branch`.
4. Chọn branch `main` và thư mục `/root`.
5. Lưu lại, GitHub sẽ tạo URL Pages cho site.
