# Slide web lộ trình thuật toán Level 0-3

Web tĩnh HTML/CSS/JS thuần, trình bày các module lý thuyết trong roadmap:

- Level 0: 6 chủ đề nền tảng.
- Level 1: 13 chủ đề thuật toán cơ bản.
- Level 2: 8 chủ đề thuật toán quan trọng, kèm danh sách OJ Level 2.
- Level 3: 13 chủ đề thuật toán cao cấp.

Mỗi chủ đề có định nghĩa, lý thuyết chi tiết, giải thích vì sao thuật toán hoạt động, phương pháp sử dụng, 2 ví dụ áp dụng, mã giả và code C++17 mẫu. Riêng Level 2 có thêm danh sách 202 bài duy nhất từ roadmap OJ, bộ lọc theo dạng bài, sample input/output thật từ đề, và hướng dẫn từng bài. Code C++17 trong từng bài Level 2 chỉ hiện khi bài có nhãn "Đáp án đã kiểm", nghĩa là lời giải riêng đã được compile và chạy sample.

Một số phần được tham khảo và diễn giải lại từ VNOI Wiki, đặc biệt là các bài về đệ quy và quay lui, tham lam, tổng quan cấu trúc dữ liệu, mảng và danh sách liên kết, bảng băm.

## Chạy local

Mở trực tiếp file:

```text
index.html
```

## Deploy GitHub Pages

1. Push repo này lên GitHub.
2. Vào `Settings` -> `Pages`.
3. Chọn `Deploy from a branch`.
4. Chọn branch `main` và thư mục `/root`.
5. Lưu lại, GitHub sẽ tạo URL Pages cho site.
